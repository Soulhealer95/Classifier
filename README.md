# Classifier
Simple python script that leverages sklearn algorithms to compare the same dataset to binary classify them.

Outputs the error for each algorithm tested

# Usage
```
$ python classifier.py
results for lda:
                                error count: 474
                                error rate: 47.353%
results for near:
                                error count: 493
                                error rate: 49.251%
results for bayes:
                                error count: 479
                                error rate: 47.852%
```
